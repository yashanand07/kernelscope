# KernelScope Philosophy

Version: 0.1

KernelScope is not a code search engine.

KernelScope is not a call graph generator.

KernelScope is not a RAG application.

KernelScope reconstructs semantic runtime behaviour from static source.

---

## Principles

1. Deterministic before probabilistic.

2. Runtime over syntax.

3. Every edge carries semantic meaning.

4. Never guess.

5. Prefer omission over incorrect reconstruction.

6. Semantic richness is more valuable than graph size.

7. The LLM explains facts.
   It never invents them.

---

These principles guide every architectural decision within KernelScope.
